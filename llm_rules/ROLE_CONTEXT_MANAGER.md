## Context Strategies

### Sliding Window
```typescript
class ContextWindow {
  maxTokens: number;
  content: Message[];
  
  addMessage(message: Message) {
    this.content.push(message);
    this.trimToFit();
  }
  
  trimToFit() {
    while (this.getTokenCount() > this.maxTokens) {
      this.content.shift();
    }
  }
}
```text
